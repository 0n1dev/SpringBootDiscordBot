# Spring Boot Discord Bot

> 기존에 NodeJS로 개발한 Discord Bot을 Spring Boot 환경으로 컨버팅하기 위해 개발.

- `Language`: Java 11
- `Framework`: Spring Boot 2.5.1
- `Dependencies`: JDA, Log4j2
- `Tool`: Lombok
- `Builder`: Gradle
---
- `VPS`: AWS Lightsail
- `VCS`: GitHub
- `CI/CD`: Jenkins
